# WeatherFetcher
Application to fetch data from OpenWeather API

## Features:
- Intuitive cli
- User can select city
- Get current weather
- Stream 5-day forecast data (3-hour step)
- Current weather is cached, and valid for 10 mins

## How to Run
### Start up Server
1. `cd server`
2. `go run .`
### Start up Client
1. `cd client`
2. `go run client`
